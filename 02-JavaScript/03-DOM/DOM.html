DOM - Document Object Model

DOM представялет любую HTML-страницу как сложный объект, который агрегирует в себя другие объекты - элементы страницы. DOM объединяет все элементы страницы в дерево объектов, корневым объектом которого является 'document'. Этот объект всегда неявно создается в любом JavaScript-коде.


У каждого элемента на странице есть 4 метода:

  • append(element) - добавляет элемент в конец ноды, а именно перед закрывающим дескриптором. Node - узел дерева.
  • prepend(element) - добавляет элемент в начало ноды, а именно после открывающего дескриптора.
  • before(element) - добавляет элемент перед открывающим дескрипотром.
  • after(element) - добавляет элемент после закрывающего дескриптора
