DOM - Document Object Model

DOM представялет любую HTML-страницу как сложный объект, который агрегирует в себя другие объекты - элементы страницы. DOM объединяет все элементы страницы в дерево объектов, корневым объектом которого является 'document'. Этот объект всегда неявно создается в любом JavaScript-коде.
